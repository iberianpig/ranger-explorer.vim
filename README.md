## installation
```vim
NeoBundle     'iberianpig/ranger-explorer.vim'
```

### Usage

```vim
nnoremap <silent><Leader>c :RangerOpenCurrentDir<CR>
nnoremap <silent><Leader>f :RangerOpenProjectRootDir<CR>
```
